Overview:
This is a basic e-commerce platform built using Flask, HTML, CSS, and JavaScript. The app allows users to buy and sell products, send messages, and report inappropriate users or content. The goal of the platform is to provide a simple and effective marketplace where users can engage with each other and manage their transactions.

Features:
1. User Registration & Login - Users can sign up and log in to the platform.
2. Buying and Selling - Users can list products for sale and browse available products.
3. Messaging System - Users can send and receive messages from other users about specific products or transactions.
4. Reporting System - Users can report inappropriate behavior or content on the platform.
5. Admin Panel - Admin users can manage reported issues and moderate the platform.

Technologies Used:
- Backend: Flask
- Frontend: HTML, CSS, JavaScript
- Database: SQLite3 (for storing user information, products, messages, and reports)
- Other Libraries: Jinja2 (templating engine), SQLite3 (used directly for database interactions)

Folder Structure:
- /static - Contains all static files (CSS, JS, images)
- /templates - Contains all HTML templates
- /app.py - Main Flask application file
- /models.py - Contains database models (using SQLite3)
- /forms.py - Contains Flask-WTForms for handling user inputs

Usage:
1. Users can register an account and log in to access the marketplace.
2. Products can be added by filling out a form and selecting images.
3. Messaging functionality allows communication between buyers and sellers.
4. Users can report other users or items that violate platform policies.
